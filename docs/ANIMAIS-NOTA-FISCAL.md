# Como os Animais Aparecem na Nota Fiscal

## Seção "Itens Adicionados"

Cada animal (bovino) da nota fiscal aparece como um **card editável** na seção **"Itens Adicionados"**, exibindo:

| Campo | Descrição |
|-------|-----------|
| **Tatuagem** | Identificação do animal (ex: M 1815 - com espaço entre letra e número) |
| **Sexo** | Macho ou Fêmea |
| **Era** | Faixa etária (ex: 13/24, 0/3, 18/24) |
| **Raça** | Ex: Mestiça, Nelore |
| **Local** | Pardinho ou Rancharia (Boletim) |
| **Peso (kg)** | Opcional |
| **Valor Unitário** | R$ formatado (ex: R$ 1.670,00) |

## Funcionalidades

- **Editar**: Todos os campos podem ser alterados clicando diretamente no card
- **Excluir**: Botão de lixeira no canto direito de cada animal
- **Aplicar a Todos**: Permite aplicar Valor, Era ou Local em lote aos itens selecionados (checkbox)
- **Total**: Soma dos valores unitários exibida em verde no topo

## Modo Receptoras

Quando "É Receptoras" está marcado, os animais aparecem no mesmo formato, com Tatuagem no padrão "M 1815" (letra + espaço + número).
